# 🏀 파라볼릭 슛: 포물선의 기적 (Parabolic Shoot)

이차함수와 포물선의 원리를 배우는 웹 기반 농구 슈팅 게임입니다. 
HTML, CSS, JavaScript(Canvas API)를 사용하여 제작되었습니다.

## 🎮 게임 방법
농구공이 날아가는 궤적은 다음 이차함수 꼭짓점 형식을 따릅니다.
**y = a(x - p)² + q**

화면 하단의 슬라이더를 조작하여 변수를 맞추고 골을 넣으세요!
* **곡률 (a):** 숫자가 작아질수록(음수) 포물선이 더 좁고 가파르게 휘어집니다.
* **최고점 위치 (p):** 포물선의 꼭짓점(가장 높은 곳)이 앞으로 갈지 뒤로 갈지 정합니다.
* **최고점 높이 (q):** 포물선이 얼마나 높이 솟아오를지 정합니다.

## 🚀 실행 방법
1. 이 리포지토리를 클론(Clone)하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 엽니다.
3. GitHub Pages를 활성화하면 웹 호스팅을 통해 즉시 플레이할 수 있습니다!
