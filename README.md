# 🏀 파라볼릭 슛 (Parabolic Shoot)

이차함수의 꼭짓점 형식($y = a(x - p)^2 + q$)을 활용하여 농구 골을 넣는 학습형 시뮬레이션 게임입니다.

## ✨ 주요 기능
- **수학적 제어:** 중앙축($p$), 최고점 높이($q$), 곡률($a$)을 직접 조절합니다.
- **클램핑 시스템:** 공이 화면 밖으로 나가지 않도록 경계 처리가 되어 있습니다.
- **예상 궤적 힌트:** 발사 초기 궤적만 표시하여 난이도와 학습 효과를 높였습니다.
- **골인 피드백:** 골인 성공 시 화려한 폭죽 효과와 메시지가 출력됩니다.

## 🕹️ 조작법
1. 슬라이더를 움직여 포물선의 모양과 위치를 결정합니다.
2. `SHOOT` 버튼을 눌러 결과를 확인합니다.
3. 골대에 정확히 들어가도록 방정식을 수정해보세요!
